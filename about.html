Accessibility Statement

We are committed to making its website accessible in accordance with the Web Content Accessibility Guidelines (WCAG) 2.2 Level AA. We believe that the internet should be available and accessible to anyone, regardless of ability.

Compliance Status

This website is partially compliant with WCAG 2.2 Level AA. We have implemented the following features to improve accessibility:

  • Semantic Structure: Using proper HTML5 tags to assist screen readers.

  • Color Contrast: We maintain high contrast ratios for text readability.
  • Keyboard Navigation: The site is navigable using only a keyboard.
